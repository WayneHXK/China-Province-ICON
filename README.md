中国地图中，每个省份的ID都是字母，对照表请参考“省份拼音对照.csv”.<br>
“省份图标 Province SVG ICON” 中的最后一列，已经在SVG代码前增加了“data:image/svg+xml;utf8,”。<br>
相比较于度量值，好处是在有很多重复项的图标视觉对象中，可以正常显示；当然缺点也是更改颜色没有度量值灵活。<br>
记得将该列的“数据类别”设置为“图像URL”。<br>
